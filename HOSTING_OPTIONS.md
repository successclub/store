# 호스팅 서비스 옵션 비교

## 현재 프로젝트 요구사항
- Node.js (버전 14 이상)
- SQLite3 (네이티브 모듈 컴파일 필요)
- 환경 변수 설정
- 포트 설정
- 정적 파일 제공

## 추천 호스팅 옵션

### 1. **Railway** ⭐ 추천
- **가격**: 무료 티어 + 유료 ($5/월부터)
- **장점**:
  - 사용하기 쉬운 인터페이스
  - 자동 배포 (Git 연동)
  - 무료 티어 제공 (제한적)
  - SQLite3 지원
  - 환경 변수 설정 쉬움
- **단점**: 무료 티어는 제한적
- **URL**: https://railway.app

### 2. **Render** ⭐ 추천
- **가격**: 무료 티어 + 유료 ($7/월부터)
- **장점**:
  - 무료 티어 제공
  - 자동 배포
  - SSL 자동 제공
  - 사용하기 쉬움
- **단점**: 무료 티어는 15분 비활성화 후 슬립 모드
- **URL**: https://render.com

### 3. **Heroku** (유료 전환)
- **가격**: $5-7/월 (무료 티어 종료)
- **장점**:
  - 매우 안정적
  - 널리 사용됨
  - 좋은 문서
- **단점**: 무료 티어 없음
- **URL**: https://www.heroku.com

### 4. **AWS (Amazon Web Services)**
- **가격**: 사용량 기반 (약 $5-15/월)
- **장점**:
  - 매우 안정적
  - 확장성 좋음
  - 무료 티어 (12개월)
- **단점**: 설정이 복잡함
- **URL**: https://aws.amazon.com

### 5. **DigitalOcean App Platform**
- **가격**: $5/월부터
- **장점**:
  - 안정적
  - 간단한 설정
  - 좋은 성능
- **단점**: 무료 티어 없음
- **URL**: https://www.digitalocean.com

### 6. **Vercel** (프론트엔드 중심)
- **가격**: 무료 + 유료
- **장점**:
  - 무료 티어
  - 매우 빠름
  - 자동 배포
- **단점**: Node.js 서버리스 함수 중심 (SQLite 제한적)
- **URL**: https://vercel.com

### 7. **Fly.io**
- **가격**: 무료 티어 + 유료
- **장점**:
  - 무료 티어
  - 전 세계 배포
  - SQLite 지원
- **단점**: 설정이 다소 복잡
- **URL**: https://fly.io

- **장점**: 한국어 지원
- **단점**: Node.js 지원 확인 필요

### 3. **NHN Cloud**
- **가격**: 사용량 기반
- **장점**: 한국 기업, 안정적
- **단점**: 가격이 높을 수 있음
- **URL**: https://www.nhncloud.com

## 주의사항

### SQLite3 지원 확인 필수!
모든 호스팅 서비스에서 SQLite3가 작동하는지 확인하세요:
- 네이티브 모듈 컴파일 지원 필요
- 파일 시스템 쓰기 권한 필요

### 무료 티어 제한사항:
- 트래픽 제한
- 메모리 제한
- 슬립 모드 (비활성화 시)
- 데이터베이스 크기 제한
